📊 Student Attendance System (DBMS Project)

📌 Description

This project is a database-based system developed to manage student attendance efficiently. It replaces manual attendance methods with a digital solution.

🎯 Objectives

- Reduce manual errors
- Improve data accuracy
- Maintain data integrity using constraints

🏗️ Features

- Add student details
- Add subject and faculty
- Mark attendance
- View attendance records

🗄️ Database Tables

- Student
- Faculty
- Subject
- Attendance

⚙️ Technologies Used

- SQL (MySQL / Oracle)
- DBMS Concepts (ER Model, Normalization)

▶️ How to Run

1. Open MySQL / Oracle
2. Run the "attendance.sql" file
3. Execute queries
